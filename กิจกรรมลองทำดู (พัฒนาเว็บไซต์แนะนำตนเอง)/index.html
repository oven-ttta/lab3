About me

ผมเป็นนักเรียนชั้นมัธยมศึกษาปีที่ 6 โรงเรียนตรีประชาพัฒนศึกษา จังหวัดชัยภูมิ รูปแบบ Home School และเป็น Backend Developer ที่มีความหลงใหลในการสร้าง Web Application และชื่นชอบในการเขียน Program ผมมีประสบการณ์ในการทำงานและทักษะเกี่ยวกับ JavaScript, Python, React, Next.js, Nest.js, Node.js, Postman, PostgreSQL, MySQL, Docker, FastAPI, HTML, CSS, Figma และ Git ผมเป็นคนเรียนรู้เร็วและกำลังมองหาวิธีเพิ่มพูนความรู้และทักษะอยู่เสมอ ปัจจุบันผมอยู่ภายใต้การดูแลโครงการ The Tiger Team Academy และผมชื่นชอบทำงานร่วมกับผู้อื่นพร้อมทั้งมั่นศึกษาอยู่เป็นประจำเพื่อที่จะพัฒนาตัวเองและพัฒนาสิ่งรอบข้างให้ดีขึ้นอยู่เสมอ
